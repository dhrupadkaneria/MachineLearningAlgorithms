The folder contains 3 files. 

tuple.java -> It describes the tuple and its basic functions.
Assignment1.java -> This file has all the calculation. The file reading, distance calculation, finding the nearest neighbour, classifying etc.
Client.java -> This file has the main program. It reads the first file and iterates through all the possible combination and for all the values of K (Nearest Neighbour).

How to run the program: 
If you are using command prompt, 
Step 1: javac Client.java
Step 2: java Client
